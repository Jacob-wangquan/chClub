  • 产品经理

    岗位描述：

    1、基于对业务流程的梳理，负责供应开发平台的产品规划、设计和产品生命周期管理（从概念提出到产品发布）；

    2、结合用户需求和产品技术架构，完成新产品/功能的概念设计和原型展示；

    3、分析项目、用户需求，分析竞争对手动态和市场动态，规划产品路线图；

    4、在技术开发前，完成交付给开发和测试团队产品需求文档，功能定义，用户界面，交互设计等详细设计文档；

    5、协调在产品生成过程中的各方利益，保证产品在满足用户需求的前提下有相对稳定及可持续发展过程；

    6、领导、执行产品线的生命周期管理，包括产品组合策略、市场研究、产品设计和原型设计、产品技术预研和方案，跟踪产品开发进展，产品评审发布，产品上线后改进等相关工作；

    7、撰写维护产品及项目文档，如用户使用手册。

    岗位要求：

    1、本科及以上，理工科专业优先；

    2、2年以上电商产品经验，有电商开发平台经验者优先；

    3、良好的学习能力、适应能力，及较强的问题分析、处理能力;

    4、良好的团队合作者，较强的抗压能力，善于沟通。

  • PHP工程师

    岗位描述：

    1、负责公司项目开发（微信公众号等）升级与维护；

    2、根据需求设计功能实现方法方式；

    3、按时按质完成分配的开发任务，并及时反馈给直接领导和需求方；

    4、编写开发文档；

    岗位要求：

    1、计算机相关专业，大专及以上学历,有相关工作经验者优先；

    2、熟悉 html / div/ css 源码编写，熟悉javascript / ajax 编程，熟练使用json / xml等进行前后台数据交互；

    3、精通PHP,熟悉think php，编程风格良好, 可独立做PHP开发；

    4、熟悉微信公众平台开发；

    5、熟悉MySQL数据库，拥有优秀数据库设计能力；

    6、具有良好的编码习惯，关注性能和安全，关注代码的质量；

    7、工作认真负责，有团队协作精神，具有良好的沟通能力，具备敬业负责的精神，有工作热情，能够按时完成公司交给的设计及编程任务；
